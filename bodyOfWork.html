Banana Finder

Banana Finer is the result of our first group project. Working with my class mates we wanted to create an application that solved the problem of ear worms, a song or lyric that gets stuck in your head but you cant remember the full song.
This application uses Chart Lyrics API to search Users lyrics and return results that contain those Lyrics.

The technical elements I was responsible for in this project included the UI/UX components of the site, creating the style guides, giving stater code for the basic javaScript functionality and acting as a technical support guide to trouble shoot a range of issues from branching to console debugging.

We went for a clean simple search screen that created a clean user experience.

The style guide included how we wanted the site to look, but also the technologies we wanted to include with specific guidelines to make sure code was readable for anyone working on the project.


The full version is currently viewable on Github pages and the repository is viewable by going to our shared Github profile.
